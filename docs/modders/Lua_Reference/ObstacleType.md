# ObstacleType

Obstacle categories used by SpellObstacleDescriptor `obstacleType`.

| Key | Value | Description |
|---|---|---|
| `usual` | 0 | Small, randomly placed battlefield obstacles placed on start of the battle. |
| `absolute` | 1 | Large obstacle blocking predefined part of the battlefield. At most one such obstacle is present on map. |
| `spellCreated` | 2 | Obstacle placed by a spell (Land Mine, Quicksand, Fire Wall, ‚Ä¶). |
| `moat` | 3 | Town-siege moat tiles. |
