# AimType

Targeting modes a spell can require: none, creature, obstacle, location.

| Key | Value | Description |
|---|---|---|
| `nothing` | 0 | Spell takes no target - massive spells that cast directly when selected. |
| `creature` | 1 | Spell targets one or more units on the battlefield. |
| `obstacle` | 2 | Spell targets an existing obstacle. |
| `location` | 3 | Spell targets one or more battlefield hexes (no unit required). |
