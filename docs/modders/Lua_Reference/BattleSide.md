# BattleSide

Battlefield side identifiers: none / attacker / defender.

| Key | Value | Description |
| --- | ----- | ----------- |
| `none` | -1 | Not associated with any side. |
| `attacker` | 0 | The attacking army that initiated the battle. Equivalent to `left` |
| `defender` | 1 | The defending army. Equivalent to `right` |
| `left` | 0 | Army that starts on the left side of the battlefield. Equivalent to `attacker`. |
| `right` | 1 | Army that starts on the right side of the battlefield. Equivalent to `defender`. |
